이 문서는 블로그에서 지원하는 Markdown 문법을 한곳에서 확인하기 위한 공개 쇼케이스입니다.

## Basic Markdown

문단, **굵게**, *기울임*, 목록을 사용할 수 있습니다.

- 첫 번째 항목
- 두 번째 항목
- 세 번째 항목

## Wiki Links

위키링크는 글 파일 이름을 기준으로 연결됩니다.

[[사용 안내]]
[[한국어 링크도 가능합니다|한국어 링크 예시]]

## Highlight

검색하거나 읽을 때 눈에 띄게 남기고 싶은 부분은 ==이렇게 표시==할 수 있습니다.

## Callout

> [!note] 참고
> 콜아웃은 Obsidian 스타일 문법을 지원합니다.

## YouTube

유튜브가 필요한 글에서는 프론트매터에 `media: youtube`를 선언하고 아래 directive를 사용합니다.

```md
---
base:
  - media
media: youtube
---

::youtube{id="Q01YOfmeK2A"}
```

## Audio

오디오가 필요한 글에서는 프론트매터에 `media: audio`를 선언하고 아래 directive를 사용합니다.

```md
---
base:
  - media
media: audio
---

::audio{src="https://example.com/audio.mp3"}
```

## Chapters

챕터는 2단계 제목의 시작 부분에 타임스탬프를 둡니다.

```md
## 0:00 Intro
## 1:30 Main section
```

## Cues

큐는 문단 앞에 재생 버튼 기호와 타임스탬프를 둡니다.

```md
▶ 0:05 This paragraph is tied to the media timestamp.
```
