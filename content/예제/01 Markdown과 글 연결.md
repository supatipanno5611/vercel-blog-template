이 글은 기본 Markdown 표현과 이 웹사이트가 추가로 제공하는 연결 기능을 보여 줍니다. [[예제/영상과 구간 이동|영상 예제]]와 [[예제/오디오와 자막|오디오 예제]]로도 이동할 수 있습니다.

## 표준 Markdown과 GFM

문단에는 **굵은 글씨**, *기울임 글씨*, `inline code`, [외부 링크](https://commonmark.org/)를 사용할 수 있습니다.

| 기능 | 작성 예 | 화면 결과 |
| --- | --- | --- |
| 체크 목록 | `- [x] 완료` | 체크 표시가 있는 목록 |
| 취소선 | `~~취소~~` | ~~취소된 문장~~ |
| 표 | 파이프 구분 | 지금 보고 있는 표 |

- [x] Markdown 글을 작성한다.
- [ ] 필요한 주제어를 추가한다.

## 강조와 알림 상자

일반 Markdown에 더해 `==중요한 구절==`처럼 작성하면 ==강조 표시==가 적용됩니다.

> [!note] 알림 상자 예제
> `[!note]`, `[!tip]`, `[!warning]` 같은 표식을 인용문 첫 줄에 넣으면 종류에 맞는 알림 상자로 표시됩니다.

> [!warning] 게시 전 확인
> 일반 글에는 `date`와 `topics` frontmatter가 필요하며, 파일명은 공개되는 제목으로 사용됩니다.

## 목차와 헤딩 링크

미디어가 없는 일반 글은 이 글처럼 헤더에서 본문 목차를 열 수 있습니다. `##`부터 `####`까지의 제목 옆 링크 복사 버튼으로 특정 단락의 주소를 공유할 수도 있습니다.

### 검색과 주제어

본문, 제목, 주제어는 검색 대상입니다. 하단의 `기능 예제` 또는 `Markdown` 주제어를 누르면 해당 주제를 공유하는 글을 탐색할 수 있습니다.

### 위키 링크와 백링크

`[[예제/영상과 구간 이동|영상 예제]]`처럼 작성한 위키 링크는 내부 링크가 됩니다. 이 글이 링크한 대상 글의 하단에는 이 글이 백링크로 표시됩니다.
